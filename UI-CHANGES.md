# Pembaruan UI

- Logo KKN baru digunakan pada halaman login, navbar, dan favicon.
- Tampilan login diperbarui dengan identitas visual merah–oranye yang selaras dengan logo.
- Tipografi, warna, jarak, tombol, form, kartu, notifikasi, dan tabel dirapikan.
- Teks antarmuka diperjelas dan diseragamkan dalam bahasa Indonesia.
- Navbar seluler kini memakai menu ringkas yang mudah disentuh.
- Tabel riwayat, dashboard admin, dan laporan berubah menjadi kartu yang mudah dibaca di layar kecil.
- Tampilan profil dan status absensi dibuat lebih terstruktur.
- Dependensi `iconv-lite` ditambahkan agar proses build dan modul PDFKit tidak menampilkan peringatan modul hilang.

## Validasi

- `npm run typecheck` berhasil.
- `npm run build` berhasil.
